## Features
- **Match Scoring:** Uses custom SVG charts to visually track how well a resume aligns with a job description (0–100%).
- **Keyword Gap Tracking:** Lists matching and missing technical skills and tools side-by-side.
- **Bullet Point Suggestions:** Automatically rewrites weaker resume descriptions into impact-focused lines tailored to the role.
- **First Impression Feedback:** Generates a realistic summary paragraph outlining how a hiring manager might initially perceive the profile.
- **Secure API Routing:** Processes API requests through Vercel Serverless Functions so that sensitive API keys are never exposed to the frontend browser.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Layouts & Keyframe Animations), Vanilla JavaScript (Async/Fetch)
- **Backend:** Node.js, Vercel Serverless Functions
- **AI Core:** Anthropic API (`claude-3-5-sonnet`)
